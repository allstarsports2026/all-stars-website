# Allstar Sports Apparel - Design System

## Brand Analysis
The Allstar brand identity is high-energy, athletic, and premium. Based on the logo analysis, the following core visual elements have been identified:

- **Primary Motif**: The Star, representing excellence and "all-star" performance.
- **Typography Style**: Bold, italicized/slanted, and heavy-weight fonts that convey speed and power.
- **Color Logic**: A high-contrast "Power Palette" of Red, Black, and White/Silver.

## Design Tokens (CSS Variables)

### Color Palette
| Token | Hex | Usage |
|-------|-----|-------|
| `--primary` | `#D90429` | Main brand red, used for buttons, links, and accents. |
| `--secondary` | `#161A1D` | Dark charcoal/black, used for headers, footers, and cards. |
| `--background`| `#FFFFFF` | Primary background color for a clean, simple look. |
| `--foreground`| `#0B090A` | Main text color (Rich Black). |
| `--border`    | `#E5383B` | Subtle red borders to maintain brand consistency. |

### Gradients
- **Action Gradient**: `linear-gradient(135deg, #D90429 0%, #A4161A 100%)`
  - Used for large buttons or hero elements to provide a metallic, premium feel.

### Typography
- **Headings**: `Archivo` (Black weight) or similar bold grotesque fonts. Slanted `italic` variants are encouraged for a "sporty" feel.
- **Body**: `Inter` or `Plus Jakarta Sans` for clarity.

## Visual Principles
1. **High Contrast**: Use strong black and white backgrounds with the vibrant red to make elements pop.
2. **Sharp Edges**: Avoid large border-radii. Use `0px` or very small radii (`4px`) to reflect the sharp, aggressive edges of the logo.
3. **Speed & Motion**: Use slanted containers or italicized text to suggest movement.
4. **Photography**: Focus on high-quality action shots of jerseys. Use dark overlays on images where white text needs to be readable.
